# Task Manager

👉 https://task-manager-hanna.streamlit.app

A simple task manager app built with Streamlit.

## Features
- Add tasks
- Edit tasks
- Delete tasks
- Mark tasks as done
- Filter tasks (All / Active / Done)
- Search tasks
- Sort by title, deadline, and priority
- Set priority and deadline

## Tech stack
- Python
- Streamlit
- JSON

## How to use
Open the app using the link above and manage your tasks directly in the browser.

## Notes
This is a beginner-friendly project created to practice Python, Streamlit, Git, and GitHub.
